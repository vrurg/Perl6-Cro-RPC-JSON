NAME
====

`Cro::RPC::JSON::BatchResponse` - container of a list of responses to a batch request.

METHODS
=======

<add(Cro::RPC::JSON::MethodResponse $resp)>
-------------------------------------------

Adds a new response object to the list

`responses(--` Seq:D)>
----------------------

Returns all response objects.
